# Section Definition Of Done

A section is `ready-for-assembly` only when:

- [ ] it can be understood without chat history
- [ ] it has a clear reader outcome
- [ ] it explains why the topic matters
- [ ] procedural topics include concrete steps
- [ ] artifacts are named exactly
- [ ] at least one example is present when examples materially improve understanding
- [ ] failure modes are listed
- [ ] links to prerequisites and downstream sections are correct
- [ ] each entity is introduced before it is used in instructions
- [ ] sensitive-origin wording scan is clean
- [ ] incomplete marker scan is clean
- [ ] review notes are either addressed or recorded
