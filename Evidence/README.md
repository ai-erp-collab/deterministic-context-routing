# Evidence

Documents in this folder are evidence *about* the methodology — reports on
how well its mechanisms actually perform when tested — as opposed to
`Derivatives/`, which are reader-facing documents generated *from* the
methodology text.

## Rules

- Every document must state its method and its limitations, not just its
  results.
- If a test was run by this project's own author rather than an
  independent third party, say so plainly in the document itself — do not
  imply outside validation that didn't happen.
- No claim without a number behind it.
- Follow the same anonymization rule as the rest of this repository: no
  real project, product, or person names — use placeholders (e.g.
  "Workspace A").
